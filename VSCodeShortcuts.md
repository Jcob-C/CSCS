# VSCode Shortcuts

## **1. File & Tab Management**

| Action                | Windows/Linux      | Mac                         |
| --------------------- | ------------------ | --------------------------- |
| New file              | `Ctrl + N`         | `Cmd + N`                   |
| Open file             | `Ctrl + O`         | `Cmd + O`                   |
| Save file             | `Ctrl + S`         | `Cmd + S`                   |
| Save all              | `Ctrl + K S`       | `Cmd + Option + S`          |
| Close file/tab        | `Ctrl + W`         | `Cmd + W`                   |
| Close all tabs        | `Ctrl + K W`       | `Cmd + K W`                 |
| Reopen closed tab     | `Ctrl + Shift + T` | `Cmd + Shift + T`           |
| Switch between tabs   | `Ctrl + Tab`       | `Cmd + Option + → / ←`      |
| Navigate back/forward | `Alt + ← / →`      | `Cmd + - / Cmd + Shift + -` |

---

## **3. Editing & Selection**

| Action            | Windows/Linux             | Mac                          |
| ----------------- | ------------------------- | ---------------------------- |
| Cut line          | `Ctrl + X`                | `Cmd + X`                    |
| Copy line         | `Ctrl + C`                | `Cmd + C`                    |
| Delete line       | `Ctrl + Shift + K`        | `Cmd + Shift + K`            |
| Move line up/down | `Alt + Up / Down`         | `Option + Up / Down`         |
| Duplicate line    | `Shift + Alt + Up / Down` | `Shift + Option + Up / Down` |
| Select word       | `Ctrl + D`                | `Cmd + D`                    |
| Select line       | `Ctrl + L`                | `Cmd + L`                    |
| Expand selection  | `Shift + Alt + Right`     | `Shift + Option + Right`     |
| Shrink selection  | `Shift + Alt + Left`      | `Shift + Option + Left`      |
| Toggle comment    | `Ctrl + /`                | `Cmd + /`                    |
| Block comment     | `Shift + Alt + A`         | `Shift + Option + A`         |

## **5. Searching & Navigation**

| Action           | Windows/Linux      | Mac                |
| ---------------- | ------------------ | ------------------ |
| Go to file       | `Ctrl + P`         | `Cmd + P`          |
| Go to symbol     | `Ctrl + Shift + O` | `Cmd + Shift + O`  |
| Go to definition | `F12`              | `F12`              |
| Peek definition  | `Alt + F12`        | `Option + F12`     |

| Find             | `Ctrl + F`         | `Cmd + F`          |
| Find next        | `F3`               | `Cmd + G`          |
| Find previous    | `Shift + F3`       | `Cmd + Shift + G`  |

## **6. Multi-cursor / Multi-selection**

| Action                 | Windows/Linux            | Mac                        |
| ---------------------- | ------------------------ | -------------------------- |
